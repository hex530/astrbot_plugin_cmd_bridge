# AstrBot 指令转发桥接器 (Command Bridge)

本插件为 AstrBot 提供了一个强大的“指令中转”能力，允许 LLM 通过自然语言描述，自主调用系统中已安装的其他插件指令。

## 核心特性

- **自然语言交互**：用户只需描述需求，LLM 即可自动匹配并执行相应的插件指令。
- **管理员特权**：支持配置管理员 QQ，管理员可无视 white_list 限制，执行任何合法的系统指令。
- **白名单机制**：为非管理员用户提供指令白名单，确保系统安全性。
- **无缝集成**：完美兼容 AstrBot 的事件系统，模拟真实用户操作。

## 安装方法

1. 在 AstrBot 插件管理中搜索 `cmd_bridge` 或通过本仓库 URL 安装。
2. 重启 AstrBot。

## 配置项说明

| 配置项 | 说明 | 示例 |
| :--- | :--- | :--- |
| `admin_qqs` | 插件管理员 QQ 列表，支持多个 QQ 用逗号分隔。 | `123456,789012` |
| `allowed_prefixes` | 允许 LLM 代发执行的指令前缀。 | `/hapi, /status, /help` |
